#Мой учебный проект
### Description 

Это учебный проект, выполненный в рамках контрольной работы по системе контроля версий Git.
Программа на языке С выводит информацию о студенте и номере группы. 
Проект демонстрирует базовые навыки работы с ветками, коммитами и слиянием в Git. 
## Installation 

1) Клонируйте репозиторий `git clone <ссылка>`
2) Перейдите в папку проекта: `cd macOS`
3) Скомпилируйте программу:  `gcc src/main.c -o app `
## Usage 

Консольное приложение, выводящее информацию о студенте и группею 
Для компиляции и запуска программы выполните: 
gcc src/main.c -o app 
./app
